UI Helpers

PaperTrail provides an extensive suite of Handlebar helpers and block extension that make building custom interfaces simple

Document tables

The {{#document_table}} helper will render a table based on a PQL query specified in it's body, the PQL can use standard Handlebars syntax and helpers to replace parameters: e.g.

{{#document_table}}
        SELECT party_name, contact_name, email, type_service, communicated FROM 'Division/Cabinet' WHERE formNo = '{{indexes.formNo}}'
{{/document_table}}

Actions can be added to each row using {{#action}} e.g. 

{{#actions}}
   {{{action docId "delete_file" icon='trash' label="Delete" refresh=true class="btn-inline"}}}
{{/actions}}

Headers can be added using {{#header}} e.g. 

{{#header}}
   {{{create 'Division/Cabinet' formNo=indexes.formNo label="Add" }}}
{{/header}}

Which renders as:

Example: Document table with header and actions per row

{{#document_table}}
    {{#header}}
       {{{create 'Division/Cabinet' formNo=indexes.formNo }}}
    {{/header}}

    {{#actions}}
      {{{action docId "delete_file" icon='trash' label="Delete" refresh=true class="btn-inline"}}}
    {{/actions}}

    SELECT party_name, contact_name, email, type_service, communicated FROM 'Division/Cabinet' WHERE formNo = '{{indexes.formNo}}'
{{/document_table}}

Document Links

{{#document_list}} is used for rendering only a list of document links e.g.

{{#document_list view=true}}
        SELECT * FROM 'Division/Cabinet WHERE formNo = '{{indexes.formNo}}'
        {{#after}}<br/>{{/after}}
{{/document_list}}

Will render a list, when clicking on the link it will redirect the user to that document

The default behaviour can be changed to view instead of redirect using {{#document_list view=true}}

{{#before}} blocks are rendered just before the link
{{#after}} blocks are rendered after the link and are commonly used to switch from a horizontal to verical list.

{{#header}} and {{#footer}} are also supported so it can be combined with an upload action:

{{#document_list view=true}}
    SELECT * FROM 'Division/Cabinet' WHERE formNo = '{{ndexes.formNo}}'
    {{#after}}<br/>{{/after}}
    {{#footer}}
        {{{upload 'Division/Cabinet' formNo=@root.indexes.formNo class="btn-inline pr5" }}}
    {{/footer}}
{{/document_list}}

Action Button

Action buttons can be added using {{{action}}}

Note that triple {{{ unlink block helpers like {{#document_table}}, action returns HTML which needs to be escaped using the standard Handlebars syntax of 3 braces:

e.g. to render a Comment button: 

{{{action docId 'add_note'}}}

Upload Button

To render an upload button: 

 {{{upload 'Division/cabinet' formNo=indexes.formNo}}}

Standard styling can be applied to render a small upload icon only like this: 

 {{{upload 'Division/cabinet' formNo=indexes.formNo  class='btn-inline' label="&nbsp;"}}}

Linked Document
